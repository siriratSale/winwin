# Binary Option Control Console

เว็บแอป Google Apps Script สำหรับจดสถิติการเทรด Binary Option ส่วนตัว  
บันทึกผล WIN/LOSS, จำนวนเงินต่อไม้, P/L, เป้ากำไรรายวัน และดูสถิติจาก Google Sheets

## Files

- Code.gs — ฝั่ง Server ของ Google Apps Script
- Index.html — หน้า Web App

## Deploy

1. เปิด Google Apps Script
2. สร้างไฟล์ Code.gs แล้ววางโค้ดจาก repo
3. สร้างไฟล์ Index.html แล้ววางโค้ดจาก repo
4. กด Deploy > Manage deployments
5. เลือก Web app
6. Deploy เป็นเวอร์ชันใหม่
